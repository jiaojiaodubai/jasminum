## 1. 基础功能

- 中文PDF元数据抓取
- 中文转换器下载，转换器来源于 Zotero中文社区 [translators_CN](https://github.com/l0o0/translators_CN)
- 中文引用格式下载，引用格式来源于项目 Zotero中文社区 [styles](https://github.com/zotero-chinese/styles)
- 小工具
  - 语言设置
  - 中文姓名折合与合并
